# Simple ATM Simulation (Java)

This is a basic Java program that simulates an ATM.  
It allows the user to:
- Check balance  
- Deposit money  
- Withdraw money  

---

## How to Run
1. Compile the program:
   ```bash
   javac Main.java
````

2. Run the program:

   ```bash
   java Main
   ```

```
